# naverQR

기존 네이버 어플의 전자출입증명서를 개선한 어플리케이션.
기존 어플은 QR코드를 키기위해 터치 횟수가 불필요하게 많았고 이를 단 한번으로 개선해보고자 다음과 같은 어플을 만듦 어플을 키기만하면 불필요한 터치 없이 
바로 QR코드가 나와 간편하게 전자출입인증을 할 수 있다.
